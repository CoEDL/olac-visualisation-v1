Introduction

OLAC, the Open Language Archives Community, is an international partnership of institutions and individuals who are creating a worldwide virtual library of language resources by: (i) developing consensus on best current practice for the digital archiving of language resources, and (ii) developing a network of interoperating repositories and services for housing and accessing such resources.

The visualisation

The application is a map based visualisation of the data at the language archives website. Interact with the map to find the languages in a specific region or browse via country using the side panel.

The app

The app is visible @ http://mlr-au.github.io/olac-visualisation/app/#/. In this version the data scraped from the language archives site is baked in however it is unlikely to ever be updated as this instance is for demonstration purposes only.
